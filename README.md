Prescription-Server | 医疗处方生成系统

本系统是一个基于 Node.js/Bun 和 XeLaTeX 的医疗处方辅助排版工具。

🚀 快速开始
---
环境要求
XeLaTeX (必须安装并添加到系统环境变量 PATH)

📦 构建
---

安装 [bun](https://bun.com/)，然后：

```bash
bun run build:all
```

（二进制文件自动镶嵌模板文件）
